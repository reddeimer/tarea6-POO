## Ejercicio 1

### Descripción del ejercicio:

> El programa usa atributos privados para guardar datos 
> del solicitante como nombre, cédula, ingresos, vehículos y 
> residencia, aplicando encapsulamiento para proteger la información.
> También se utiliza un constructor para crear el objeto con valores 
> iniciales. Se aplican validaciones para evitar datos incorrectos y 
> se usa lógica condicional para verificar si cumple los requisitos 
> del subsidio.

## Ejercicio 2

### Descripción del ejercicio:

> El programa se basa en un sistema bancario aplicando Programación Orientada a
> Objetos para gestionar distintos tipos de cuentas y calcular automáticamente sus
> intereses mensuales además de distintos calculos generales de un sistema bancario,
> a su vez se implementaron atributos privados, protegidos y públicos junto con
> los pilares de la programación (encapsulamiento, herencia, polimorfismo y
> abstracción), en donde cada clase tiene una implementación diferente de métodos, 
> propio cálculo de interés. Se aplicaron validaciones para evitar que no se
> mantenga el correcto flujo lógico que se quiere tener para el problema, especialmente
> en la creación de los métodos para realizar el retiro y depósito en una cuenta.

### Ejercicio Hecho por Diego Almeida.

## Instrucciones para ejecutar el proyecto

### Requisitos

> - ``intelliJ IDEA``  
> - ``Java JDK 25+ ``
> - ``Git ``
> - ``Acceso a GitHub ``

### Pasos

> 1. Clonar repositorio de GitHub usando gitclone y la url de este repositorio.
> 2. Abrir intelliJ y abrir proyecto seleccionando la carpeta que contiene a todas las subcarpetas.
> 3. Ejecutar los archivos Main respectivos de cada ejercicio (Para ejercicio 1 el Main Principal, Para el ejercicio 2 su propio archivo Main) usando ``SHIFT + F10`` o usando el botón ya proporcionado por IntelliJ.
